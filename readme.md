# Chem E Car code for CCNY
## Connection Diagram:
   - Brown Wire is power, connect it to 5 volts
   - Blue Wire is ground, you can connect it to any of the grounds on the arduino
   - Black wire is data, this MUST be connected to Digital Pin 2
